# Movie Genre Prediction App

## Description
This application predicts the genre of a movie based on its plot summary using NLP and Machine Learning.

## Files
- app.py : model training and prediction
- streamlit.py : Streamlit web application

## Dataset
Dataset not uploaded due to size limitations.
Format:
train_data.txt - id ::: title ::: plot_summary ::: genre
test_data.txt  - id ::: title ::: plot_summary

## How to Run
pip install -r requirements.txt
streamlit run streamlit.py
